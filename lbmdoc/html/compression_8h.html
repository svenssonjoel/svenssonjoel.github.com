char* lbm_compress ( char *  string,

Compress lisp code. This function is not suitable for execution on a microcontroller.

Parameters
stringString to compress. This string cannot be constant. The compression algorithm will make destructive updates to the input string.
res_sizeThe resulting, compressed, size is stored in this variable.
Returns
A pointer to compressed data allocated using malloc. The user is responsible for freeing this memory.
