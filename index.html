Projects

arbb-vm

ArBB virtual machine bindings are available here

EmbArBB

EmbArBB is a language for data parallel programming embedded in Haskell. This project implements a higher-level interface to ArBB compared to the arbb-vm bindings above (arbb-vm is used in the backend of this implementation).

libjit-hs

libjit-hs contains Haskell bindings to the GNU libJIT library. Currently work-in-progress.

RayCast

A retro style raycasting graphics engine (like in the first wolfenstein3D game) implemented in Haskell and C. I hope to make a dungeon exploration game (in the style of Eye of the Beholder) using this graphics engine at some time.

At first the code was much influenced by the book "Gardens of Imagination" but digressed quite a bit after a while. repository

Obsidian

Obsidian is a GPGPU programming language embedded in Haskell. Three different versions are available on github. For more information about Obsidian see my work webpage

Obsidian

This is the most recent version of Obsidian. It builds on the GCDObsidian ideas but introduces global arrays. repository

GCDObsidian

The version of Obsidian that introduced Push-Arrays in the Obisidan setting.

ArrowObsidian

Obsidian with an Arrow-like interface.

MonadObsidian

Similar to ArrowObsidian but with a monad based interface.
