LispBM

LispBM is a lisp/scheme like lanuage implemented in C targeting microcontrollers such as the STM32 or the NRF52. LispBM can be "embedded" into an application running on a microcontroller to make your application scriptable using a lisp-like language.

LispBM sources can be found on GitHub.
